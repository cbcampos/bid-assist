# BidAssist Page Feature Plan

## Page-by-Page Features

### 1. LEADS
- [ ] Dropdown to filter by status (All/Hot/Warm/New)
- [ ] Click row to expand details modal
- [ ] Details: company, contact, phone, email, source, value, status, notes, created date
- [ ] Quick actions: Call, Email, Convert to Bid
- [ ] Add Lead form modal
- [ ] Edit Lead form modal
- [ ] Delete with confirmation

### 2. INVOICES
- [ ] Filter by status (All/Draft/Sent/Paid/Overdue)
- [ ] Click row to expand details
- [ ] Details: invoice #, client, line items, subtotal, tax, total, status, due date, paid date
- [ ] Actions: Mark as Paid, Send Reminder, Print, Email
- [ ] Add Invoice form
- [ ] Edit Invoice form

### 3. 811 TICKETS
- [ ] Filter by status (All/Active/Pending/Complete/Expired)
- [ ] Map link (Google Maps address)
- [ ] Click for full ticket details
- [ ] Ticket #: Alabama One-Call format
- [ ] Location details, start date, expiration, utility marks
- [ ] Add Ticket form

### 4. CHANGE ORDERS
- [ ] Link to parent project
- [ ] Status: Pending/Approved/Rejected
- [ ] Amount, description, reason
- [ ] Approve/Reject actions
- [ ] Timeline/history
- [ ] Add CO form

### 5. SUBCONTRACTORS
- [ ] Trade filter
- [ ] Status: Available/Busy/Unavailable
- [ ] Expand: full contact info, insurance info, ratings, past jobs
- [ ] Quick call/email actions
- [ ] Add Sub form

### 6. SAFETY
- [ ] Incident log with dates
- [ ] Severity: Minor/Moderate/Serious
- [ ] Details: what happened, location, personnel involved
- [ ] OSHA recordable flag
- [ ] Days since last incident counter
- [ ] Add Incident form

### 7. LICENSING
- [ ] Type: Business License, Liability, Equipment, Contractors License
- [ ] Status: Active/Expiring Soon/Expired
- [ ] Expiration date with countdown
- [ ] Certificate upload indicator
- [ ] Reminder settings
- [ ] Add License form

### 8. TEAM
- [ ] Role filter (All/Admin/Project Manager/Field/Viewer)
- [ ] Status: Active/Invited/Inactive
- [ ] Details: name, email, role, phone, last login
- [ ] Invite new user
- [ ] Edit permissions
- [ ] Deactivate user

### 9. FIELD REPORTS
- [ ] Date filter
- [ ] Project filter
- [ ] Expand: crew list, equipment used, materials, photos indicator, issues, notes
- [ ] Daily summary
- [ ] Weather conditions
- [ ] Add Report form

### 10. WEATHER (already done)
- [ ] Already has 5-day forecast
- [ ] Already has rain alert
- [ ] Could add: job site specific forecasts

## Implementation Order
1. Leads (most used)
2. Invoices (revenue)
3. 811 (compliance critical)
4. Field Reports (daily use)
5. Subcontractors (common reference)
6. Change Orders (project related)
7. Safety (occasional)
8. Licensing (occasional)
9. Team (occasional)
